📁 Linux 文件系统

一切从根开始

🌳 Linux 目录树

Linux 的文件系统是一个倒置的树状结构

/ (根目录)
├── home (用户主目录)
│ └── student (你的目录)
├── etc (配置文件)
├── bin (可执行程序)
├── usr (用户程序)
├── var (变量数据)
└── tmp (临时文件)

所有文件和目录都从 / 开始

📂 重要目录说明

  • / - 根目录，所有文件的起点
  • /home - 用户的家目录，存放个人文件
  • /etc - 系统配置文件
  • /bin - 基本命令程序
  • /usr - 用户安装的程序
  • /var - 日志和变化的数据
  • /tmp - 临时文件，重启后清空

🛤️ 路径的概念

绝对路径

从根目录 / 开始

/home/student/documents/file.txt

完整的地址，不会混淆

相对路径

从当前位置开始

./documents/file.txt
../other/file.txt

. 表示当前目录
.. 表示上级目录

⌨️ Linux 基础命令

掌握这 10 个命令，你就入门了

1️⃣ pwd - 我在哪里？

pwd = Print Working Directory（打印当前目录）

student@linux:~$ pwd
/home/student
  • 显示当前所在的目录
  • 最简单的命令，没有参数
  • 随时知道自己在哪里

2️⃣ ls - 看看有什么

ls = List（列出文件）

student@linux:~$ ls
Desktop Documents Downloads Pictures
student@linux:~$ ls -l
drwxr-xr-x 2 student student 4096 Jan 15 10:30 Desktop
  • ls - 列出文件和目录
  • ls -l - 详细信息（权限、大小、时间）
  • ls -a - 显示隐藏文件（以 . 开头）
  • ls -lh - 人类可读的文件大小

3️⃣ cd - 移动位置

cd = Change Directory（切换目录）

student@linux:~$ cd Documents
student@linux:~/Documents$ cd ..
student@linux:~$ cd /etc
student@linux:/etc$ cd ~
student@linux:~$
  • cd 目录名 - 进入某个目录
  • cd .. - 返回上一级
  • cd ~ 或 cd - 回到家目录
  • cd - - 回到上次的目录

4️⃣ mkdir - 创建目录

mkdir = Make Directory（创建目录）

student@linux:~$ mkdir my_project
student@linux:~$ mkdir -p work/2024/january
student@linux:~$ ls
my_project work
  • mkdir 目录名 - 创建新目录
  • mkdir -p - 创建多级目录
  • 目录名不要有空格，用下划线代替

5️⃣ touch - 创建文件

touch - 创建空文件或更新时间戳

student@linux:~$ touch readme.txt
student@linux:~$ touch file1.txt file2.txt file3.txt
student@linux:~$ ls
file1.txt file2.txt file3.txt readme.txt
  • touch 文件名 - 创建空文件
  • 可以一次创建多个文件
  • 如果文件已存在，更新修改时间

6️⃣ cat - 查看文件内容

cat = Concatenate（连接并显示）

student@linux:~$ cat readme.txt
Hello, Linux!
This is my first file.
  • cat 文件名 - 显示文件全部内容
  • 适合查看小文件
  • 大文件用 less 或 more

7️⃣ cp - 复制文件

cp = Copy（复制）

student@linux:~$ cp file1.txt file1_backup.txt
student@linux:~$ cp -r my_project my_project_backup
  • cp 源文件 目标文件 - 复制文件
  • cp -r - 复制目录（递归）
  • cp -i - 覆盖前询问

8️⃣ mv - 移动/重命名

mv = Move（移动）

student@linux:~$ mv file1.txt Documents/
student@linux:~$ mv old_name.txt new_name.txt
  • mv 源 目标 - 移动文件或目录
  • 也用于重命名文件
  • mv -i - 覆盖前询问
  • 不需要 -r 参数，可以直接移动目录

9️⃣ rm - 删除文件

rm = Remove（删除）⚠️ 危险命令

student@linux:~$ rm file1.txt
student@linux:~$ rm -r my_folder
student@linux:~$ rm -i file2.txt
rm: remove regular file 'file2.txt'? y
  • rm 文件名 - 删除文件
  • rm -r - 删除目录（递归）
  • rm -i - 删除前确认（推荐）
  • ⚠️ 永远不要执行 rm -rf /

🔟 man - 查看帮助

man = Manual（手册）

student@linux:~$ man ls
LS(1) User Commands LS(1)
NAME
ls - list directory contents
  • man 命令名 - 查看命令的详细说明
  • 按 q 退出
  • 按 空格 翻页
  • 最好的学习资源就在系统里

📝 10 个基础命令总结

导航

pwd, ls, cd

创建

mkdir, touch

查看

cat, man

操作

cp, mv, rm

记住这 10 个，你就能完成 80% 的日常操作

💡 实用技巧

让你的操作更高效

⌨️ Tab 键自动补全

最重要的快捷键！

student@linux:~$ cd Doc[按 Tab]
student@linux:~$ cd Documents/
  • Tab 键可以自动补全命令和文件名
  • 按两次 Tab 显示所有可能的选项
  • 大大提高输入速度，减少错误
  • 养成习惯，随时按 Tab

🔄 命令历史

不用重复输入相同的命令

  • 按上箭头 ↑查看上一条命令
  • 按下箭头 ↓查看下一条命令
  • history 命令查看所有历史
  • Ctrl + R 搜索历史命令
student@linux:~$ history
1 pwd
2 ls -la
3 cd Documents

🃏 通配符

批量操作文件的利器

student@linux:~$ ls *.txt
file1.txt file2.txt readme.txt
student@linux:~$ rm file?.txt
(删除 file1.txt, file2.txt)
  • * - 匹配任意多个字符
  • ? - 匹配单个字符
  • [abc] - 匹配括号中的任意一个字符

⚡ 常用快捷键

  • Ctrl + C - 终止当前命令
  • Ctrl + D - 退出终端（或 exit 命令）
  • Ctrl + L - 清屏（或 clear 命令）
  • Ctrl + A - 光标移到行首
  • Ctrl + E - 光标移到行尾
  • Ctrl + U - 删除光标前的内容
  • Ctrl + K - 删除光标后的内容

🔐 文件权限基础

Linux 的安全机制

📋 理解文件权限

student@linux:~$ ls -l readme.txt
-rw-r--r-- 1 student student 1024 Jan 15 10:30 readme.txt

-rw-r--r-- 权限说明：

  • - 文件类型（- 文件，d 目录）
  • rw- 所有者权限（读、写）
  • r-- 组权限（只读）
  • r-- 其他人权限（只读）

🔤 权限字母含义

📖

r (read)

读权限 - 可以查看文件内容

✏️

w (write)

写权限 - 可以修改文件

▶️

x (execute)

执行权限 - 可以运行程序

🚫

- (none)

无权限

🔧 chmod - 修改权限

chmod = Change Mode（修改模式）

student@linux:~$ chmod +x script.sh
(给文件添加执行权限)
student@linux:~$ chmod 755 script.sh
(rwxr-xr-x)
  • chmod +x 文件 - 添加执行权限
  • chmod 755 - 数字方式设置权限
  • 常用：644（文件）、755（可执行文件）

⚠️ 新手常见错误

  • 大小写敏感 - file.txt 和 File.txt 是不同的文件
  • 空格问题 - 文件名有空格要用引号或转义
  • 路径错误 - 注意当前目录位置
  • 权限不足 - 某些操作需要 sudo
  • 删除谨慎 - rm 删除的文件无法恢复
student@linux:~$ cd my folder
bash: cd: my: No such file or directory
student@linux:~$ cd "my folder"
✓ 正确
