### 第 35 页：💻 命令行体验（互动）
这是一个真实的 Linux 终端模拟器！

**如何使用：**
1. 点击输入框（或直接开始输入）
2. 输入命令后按 **Enter** 执行
3. 查看命令输出结果

**可用命令：**

| 命令 | 功能 | 示例 |
|------|------|------|
| `whoami` | 显示当前用户名 | `whoami` |
| `date` | 显示当前日期时间 | `date` |
| `pwd` | 显示当前工作目录 | `pwd` |
| `ls` | 列出当前目录文件 | `ls` |
| `echo` | 输出文本 | `echo Hello Linux` |
| `uname` | 显示系统信息 | `uname` |
| `fortune` | 随机显示一句格言 | `fortune` |
| `cowsay` | ASCII 艺术牛说话 | `cowsay 你好` |
| `clear` | 清空屏幕 | `clear` |
| `help` | 显示帮助信息 | `help` |

**趣味命令：**
- 试试 `fortune` 看看会得到什么格言
- 试试 `cowsay 我爱Linux` 让牛说话
- 试试 `echo 今天学习Linux` 输出自定义文本

**提示：**
- 如果输入错误的命令，会提示 "command not found"
- 输入 `help` 可以查看所有可用命令
- 使用 `clear` 可以清空屏幕重新开始

## 🔧 故障排除

### 页面显示不正常
- 确保使用现代浏览器（Chrome、Firefox、Edge、Safari）
- 清除浏览器缓存
- 刷新页面（Ctrl+F5 或 Cmd+Shift+R）

### 点击无反应
- 确保 JavaScript 已启用
- 检查浏览器控制台是否有错误
- 尝试重新加载页面

### 终端不工作
- 确保点击了输入框
- 检查是否在第 35 页
- 尝试输入 `help` 测试

### 模态框无法关闭
- 点击窗口外的灰色区域
- 点击右上角的 × 按钮
- 刷新页面
