# Linux 操作系统 - 第一节课 PPT

这是一份面向大一新生的《Linux操作系统》第一节课的交互式 HTML 演示文稿。

## 📁 文件说明

- `linux_intro_presentation.html` - 主演示文稿文件
- `info_database.js` - 详细信息数据库
- `README.md` - 使用说明

## 🚀 如何使用

1. 直接用浏览器打开 `linux_intro_presentation.html` 文件
2. 使用以下方式控制幻灯片：
   - **键盘方向键** ← → 切换幻灯片
   - **空格键** 下一页
   - **鼠标点击** 屏幕右下角的按钮
   - **触摸滑动**（移动设备）左右滑动切换

## ✨ 特色功能

### 交互式内容
- 所有带有 **ℹ️ 图标**的内容都可以点击
- 点击后会弹出详细解释窗口
- 包含扩展知识、学习资源和相关网站链接

### 可视化生态系统（第 34 页）
- 交互式 SVG 图表展示 Linux 应用领域
- 点击不同的圆圈查看详细信息
- 涵盖：服务器、云计算、开发、安全、嵌入式、数据科学
- 动态连接线和悬停效果
- 中心 Linux 标志带有脉动动画

### 命令行模拟器（第 35 页）
- 真实的终端界面体验
- 支持多个 Linux 命令：
  - `whoami` - 查看当前用户
  - `date` - 显示日期时间
  - `pwd` - 显示当前目录
  - `ls` - 列出文件
  - `echo` - 输出文本
  - `uname` - 系统信息
  - `fortune` - 随机格言
  - `cowsay` - ASCII 艺术
  - `clear` - 清屏
  - `help` - 帮助信息
- 输入命令后按 Enter 执行
- 命令错误提示
- 自动滚动到最新输出

### 涵盖内容
1. **Linux 是什么** - 定义、历史、发行版
2. **Linux 能做什么** - 应用场景、实际案例
3. **Linux vs Windows** - 对比分析
4. **为什么学 Linux** - 职业发展、技术优势
5. **学习前置条件** - 需要的基础知识
6. **学习路径** - 四个阶段的学习规划
7. **AI 赋能** - 如何用 AI 工具辅助学习
8. **互动探索** - Linux 生态系统可视化地图
9. **命令行体验** - 模拟终端，实际操作命令

## 📝 自定义

如果需要修改内容：

1. **修改幻灯片内容**：编辑 `linux_intro_presentation.html` 中的 HTML
2. **添加详细信息**：在 `info_database.js` 中添加新的条目
3. **修改样式**：调整 HTML 文件中的 CSS 部分

### 添加新的可点击内容

```html
<span class="clickable" data-info="your-key">你的内容</span>
```

然后在 `info_database.js` 中添加：

```javascript
infoDatabase['your-key'] = {
    title: "标题",
    content: `
        <p>详细内容...</p>
    `
};
```
