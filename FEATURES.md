# 功能特性详解

## 🎯 核心功能

### 1. 交互式内容系统

#### 实现方式
- 使用 `data-info` 属性标记可点击元素
- JavaScript 事件监听器捕获点击
- 从 `info_database.js` 加载详细内容
- 模态框动态显示信息

#### 特点
- 30+ 个可点击的知识点
- 每个知识点包含：
  - 详细解释
  - 实际应用案例
  - 学习资源链接
  - 相关网站推荐
- 视觉提示（ℹ️ 图标）
- 悬停效果（高亮、上移）

#### 涵盖主题
- **技术概念**：开源、Unix、Linux 发行版
- **公司案例**：Google、Facebook、Amazon、AWS、阿里云
- **应用场景**：Android、特斯拉、国际空间站、Steam Deck
- **专业领域**：软件开发、数据科学、网络安全、嵌入式、超级计算机
- **职业方向**：后端开发、SRE、云计算
- **AI 工具**：GitHub Copilot、ChatGPT、Explain Shell、Warp
- **学习方法**：虚拟机搭建、项目实践、Man 手册、学习社区

### 3. 命令行模拟器（第 35 页）

#### 技术实现
- 纯 JavaScript 实现
- 命令解析器
- 输出渲染系统
- 历史记录管理

#### 终端界面
- **顶部栏**：
  - macOS 风格按钮（红、黄、绿）
  - 终端标题显示
  
- **输出区域**：
  - 300px 高度
  - 自动滚动
  - 自定义滚动条
  - 等宽字体（Courier New）
  
- **输入区域**：
  - 提示符（student@linux:~$）
  - 实时输入
  - Enter 执行

#### 支持的命令

| 命令 | 功能 | 实现 |
|------|------|------|
| `whoami` | 显示用户名 | 返回 "student" |
| `date` | 显示日期时间 | JavaScript Date 对象 |
| `pwd` | 当前目录 | 返回 "/home/student" |
| `ls` | 列出文件 | 从文件系统对象读取 |
| `echo` | 输出文本 | 返回参数 |
| `uname` | 系统信息 | 返回模拟的系统信息 |
| `fortune` | 随机格言 | 从数组随机选择 |
| `cowsay` | ASCII 艺术 | 字符串拼接 |
| `clear` | 清屏 | 清空输出区域 |
| `help` | 帮助信息 | 显示命令列表 |

#### 特殊功能
- **错误处理**：未知命令提示
- **参数支持**：echo、cowsay 支持参数
- **命令历史**：记录执行过的命令
- **自动滚动**：新输出自动滚动到底部
- **颜色高亮**：
  - 提示符：绿色
  - 输入：白色
  - 输出：浅灰色
  - 错误：红色

#### Easter Eggs
- `fortune` 有 8 条不同的格言
- `cowsay` 可以自定义文本
- ASCII 艺术牛的图案

## 📈 扩展性

### 添加新的可点击内容
1. 在 HTML 中添加 `class="clickable" data-info="key"`
2. 在 `info_database.js` 中添加对应的数据
3. 无需修改 JavaScript 逻辑

### 添加新的生态系统节点
1. 在 SVG 中添加新的 `<g>` 元素
2. 在 `ecosystemData` 中添加数据
3. 调整布局坐标

### 添加新的终端命令
1. 在 `commands` 对象中添加新函数
2. 在 `help` 命令中添加说明
3. 实现命令逻辑
